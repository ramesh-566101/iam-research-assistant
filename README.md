# IAM Research Assistant

A research data assistant that allows users to query and interpret Identity & Access Management (IAM) datasets using Large Language Models.

## Features

- Natural language queries
- User risk analysis
- SoD violation detection
- Certification review insights
- Privileged access reporting

## Tech Stack

- Python
- Jupyter Notebook
- vLLM
- Qwen3 32B
- AMD MI300 GPU
- ROCm 7

## Sample Questions

- How many active users have high-risk access?
- List users with SoD violations.
- Which department has the most privileged accounts?
- Show inactive privileged users.
